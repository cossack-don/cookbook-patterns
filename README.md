# Cookbook Patterns (functional paradigm javascript/typescript)

## Порождающие
- Строитель (Builder)
- Прототип (Clon, Prototype)
- Одиночка (Singleton)
- Абстрактная фабрика (Abstract Factory)
- Фабрика (Factory)
  

## Структурные
- Декоратор (Wrapper, Обёртка, Decorator)

## Поведенческие
- Посредник (Intermediary, Controller, Mediator)
